STATVULT-Pro is for visualizing entities and comparing them on multiple graphs.

# INSTALLATION
1. Install npm and Node JS if you don't already have them follow this [tutorial](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
2. Clone the project
```markdown
git clone https://github.com/Flame21i/STAT-VAULT.git; cd STAT-VAULT
```
4. Initilize npm
```markdown
npm install
```
6. Install the Required Dependendcies:
```markdown
npm install recharts lucide-react react-is
```
8. Navigate to the project folder and run:
```markdown
npm run dev
```
